JSON
 4. Создать внешний репозиторий c названием JSON. - на github.com "create a new repository", add discription "First part of GIT_HW_1" and enter "Create repository"
 5. Клонировать репозиторий JSON на локальный компьютер. Enter "code", copy URL > git clone https://github.com/RacooPrArt/JSON.git
 6. Внутри локального JSON создать файл “new.json”. touch new.json
 7. Добавить файл под гит. - git add .
 8. Закоммитить файл. - git commit . -m "add first file"
 9. Отправить файл на внешний GitHub репозиторий. git push
 10. Отредактировать содержание файла “new.json” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате JSON. 

vim new.json
i
{ 
	"name": "Artem",
	"age": 27,
	"number of pets": 0,
	"salary expectations": "80 000 RUB",
	"city": "Moscow"
}
Enter "Esc"
:wq

 11. Отправить изменения на внешний репозиторий.

git commit -am "update new.json"
git push

 12. Создать файл preferences.json
touch preferences.json

 13. В файл preferences.json добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате JSON.
vim preferences.json
i
{
	"favourite film": "La-La-Land",
	"favourite Series": "Big Bang Theory",
	"favourite food": "meat",
	"favourite season": "summer",
	"desired country": "Czech"
}
Enter "Esc"
:wq

 14. Создать файл sklls.json добавить информацию о скиллах которые будут изучены на курсе в формате JSON

touch sklls.json
vim sklls.json
i
{
	"linux tool": "GitBash",
	"version control tool": "Github",
	"task tracking system": "jira"
}
Enter "Esc"
:wq

 15. Отправить сразу 2 файла на внешний репозиторий.
git add .
git commit . -m "commit both files"
git push

 16. На веб интерфейсе создать файл bug_report.json.
click "add file" > "create file"

 17. Сделать Commit changes (сохранить) изменения на веб интерфейсе.
click green "commit changes"

 18. На веб интерфейсе модифицировать файл bug_report.json, добавить баг репорт в формате JSON.
click "pencil"
{
  "title": "error 404 when i try to open WebSite's Main Page",
  "steps to reproduce": "open link https://www.bugwebsite.example",
  "severity": "blocker",
  "priority": "high",
  "actual result": "error 404",
  "expected result": "open main page"
}

 19. Сделать Commit changes (сохранить) изменения на веб интерфейсе.
click "commit changes"

 20. Синхронизировать внешний и локальный репозиторий JSON
git fetch
git pull
